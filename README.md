## Progressive Repository Folder Structure

```text
java-dsa-practice/
│
├── 📁 basics/                 # Fundamentals (I/O, loops, basic math checks)
├── 📁 math_problems/          # Logic & Number Theory (Prime, bits, divisibility)
├── 📁 arrays_basics/          # Array Fundamentals (Traversals, frequency, rotations)
├── 📁 arrays_intermediate/    # Matrix & Sliding Windows (Kadane's, Two-Sum, Stock)
├── 📁 strings_basics/         # String Foundations (Anagrams, Palindromes, Prefix)
├── 📁 strings_advanced/       # Advanced Stream Parsing (Sliding Windows, Compression)
├── 📁 recursion/              # Backtracking & Trees (Subsets, Permutations, Hanoi)
├── 📁 searching_algorithms/   # Binary Search Spaces (Rotated arrays, Peak elements)
├── 📁 sorting_algorithms/     # Sorting & Partitioning (Merge, Quick, Swap Cycles)
├── 📁 stack/                  # Monotonic & Expressions (Histogram, Stock Span, Parser)
├── 📁 queue/                  # FIFO Processing (BFS state loops, Sliding max, LRU Cache)
├── 📁 linked_list/            # Pointer Mastery (Cycles detection, Merge, Reversals)
├── 📁 trees/                  # Hierarchical Structures (Traversals, BST Validations)
├── 📁 graphs/                 # Network Topology Algorithms (BFS, DFS, Dijkstra)
└── 📁 dynamic_programming/    # Optimization Mechanics (Knapsack, Subsequences)
```

> 📌 *Note: Detailed problem sheets, algorithmic breakdowns, and complexity proofs are documented individually under readme files inside each separate module directory.*
